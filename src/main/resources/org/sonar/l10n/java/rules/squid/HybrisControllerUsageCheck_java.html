This rule detects direct usage of DAOs in Controller. In Hybris Controller should only interact with Facades and DTO.

Noncompliant Code Example


@Controller
public class HybrisControllerCheckModelUsage {
...
    ProductDao productDao; // Noncompliant
...
}

Compliant Solution

@Controller
public class HybrisControllerCheckModelUsage {
...
    ProductFacade productFacade; // Compliant
}
